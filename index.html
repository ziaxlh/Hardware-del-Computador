Conociendo el Hardware del Computador

Descubre las partes que hacen posible el funcionamiento del mundo digital.

Partes Internas

Tarjeta madre (Motherboard)

Componente principal donde se conectan los demás dispositivos. Permite la comunicación entre ellos.

Memoria RAM

Almacena datos temporales para ser usados rápidamente por el procesador.

Procesador (CPU)

Es el cerebro del computador. Ejecuta instrucciones y coordina las operaciones del sistema.

Tarjeta gráfica (GPU)

Procesa gráficos y ayuda al CPU en tareas visuales intensivas.

Disco duro (HDD/SSD)

Dispositivo de almacenamiento que guarda información de manera permanente.

Fuente de Poder

Proporciona energía eléctrica estable a todos los componentes del computador.

Disipador

Enfría los componentes para evitar sobrecalentamientos y mantener el rendimiento.

Ventilador

Ayuda a mantener la temperatura adecuada dentro del gabinete.

Unidad CD-ROM

Permite leer discos compactos para acceso a datos y software.

Partes Externas

Monitor

Dispositivo de salida que muestra información visual al usuario.

Teclado

Permite introducir texto y comandos al computador.

Ratón (Mouse)

Facilita la navegación e interacción con la interfaz gráfica.

Impresora

Dispositivo que genera copias físicas de archivos digitales.

Escáner

Permite digitalizar documentos físicos para usarlos en el computador.

Altavoces

Dispositivo de salida de audio que permite escuchar sonidos del sistema.

Memoria USB

Dispositivo portátil para almacenar y transferir datos.

Webcam

Captura imágenes y video para videollamadas y grabaciones.

Micrófono

Captura sonido para grabaciones, llamadas o comandos de voz.

Proyector

Proyecta la imagen del computador en superficies grandes para presentaciones.

Joystick

Control de entrada para juegos y simuladores.
